appendToURL

  • appendToURL(url: string, data?: string | {}, options?: IQSStringifyOptions): string
    • Defined in append.ts:25

    把键值对序列化为查询字符串后拼接到指定 URL。

    example
    import { appendToURL } from '@just4/querystring/index';
appendToURL('https://abc.com/?a=1#hash', {
  b: 2,
  c: 3
}); // https://abc.com/?a=1&b=2&c=3#hash

    Parameters

    • url: string

      指定URL。

    • Optional data: string | {}

      键值对。

    • Optional options: IQSStringifyOptions

      序列化选项。

    Returns string

    处理后的 URL。
