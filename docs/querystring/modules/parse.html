  • parse(str: string, options?: Readonly<IQSParseOptions>): {}
    • Defined in parse.ts:22

    把查询字符串解析为键值对集合。

    example
    import { parse } from '@just4/querystring';
    parse('id=0&str=hello'); // { id: '0', str: 'hello' }
    parse('id=0&id=1'); // { id: ['0', '1'] }

    Parameters

      查询字符串。
