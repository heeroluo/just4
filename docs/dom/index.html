@just4/dom

@just4/dom

提供 DOM 操作接口。

特性
  • 包含常用的 DOM 操作接口，整体用法与 jQuery 相似，是 jQuery 功能的子集。
  • 元素匹配通过浏览器原生的 querySelectorAll、matchesSelector 接口实现，避免引入庞大的 Sizzle 引擎。
  • 支持 PC 和移动端所有主流浏览器（其中 IE 浏览器的最低兼容版本是 9）。
