@just4/util

本程序包主要为 Just4 的其他包提供基础的工具方法和工具类。包括：

  • 数组操作。
  • 对象操作。
  • 日期操作。
  • 类型判断。
  • 事件操作。

安装

npm i @just4/util

调用

工具函数和工具类的功能及用法都较为简单，可参阅API 文档中的代码示例。
